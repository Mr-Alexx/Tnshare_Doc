# 安装教程

1. 下载安装文件
2. 复制 chrome://extensions/ 并粘贴到地址栏，按回车键进入扩展程序页面
3. 安装插件：开发者模式 -> 刷新页面 -> 下载好的 .crx 文件拖入扩展程序
